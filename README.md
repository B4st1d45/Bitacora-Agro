# 📄 SISTEMA DE GESTION AGRICOLA - ANDROID 🚜

Aplicación móvil desarrollada en **Kotlin** diseñada para digitalizar la operación de maquinaria y mantenimientos en terreno.

---

## Funcionalidades Clave
- **Órdenes de Trabajo:** Registro dinámico de faenas, control de químicos y tiempos de operación.
- **Mantenimientos:** Gestión de servicios técnicos con alertas de horómetro.
- **Generación de Reportes:** Exportación automática de documentos PDF profesionales listos para compartir.
- **Sincronización Cloud:** Integración con Firebase Realtime Database para respaldo instantáneo.

## Tecnologías
- **Lenguaje:** Kotlin
- **Base de Datos:** Firebase Realtime Database
- **UI:** Material Design 3 (Dark Mode industrial)
- **PDF Engine:** Android PdfDocument API, PdfDocument API, Google FileProvider.
